# 212Factory
This repo is made as a part of excersise in a Data Science course

### Contents

The tesla.py file contains the "Tesla" class and the modelx.py file has the subclass "ModelX". 

In the test_tesla.py and test_modelx.py files are the pytest tests, checking that all the class elements work as expected.
The factory.py file has examples of how to code works.

### How to use

To call the class use - `tesla = Tesla('model', 'color')`

To check color `print(tesla.color)`

Other thing you can explore:

`tesla.seats_count` to check number of seats

`tesla.seats_count(number)` insert number to change seat count
